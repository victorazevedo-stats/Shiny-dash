# Dados brutos

Use esta pasta para armazenar arquivos originais baixados ou recebidos.

Sugestao de organizacao:

```text
data-raw/
|-- caged/
|-- rais/
|-- sidra/
|-- pib/
|-- arquivos_auxiliares/
```

Regras:

- nao editar manualmente os arquivos aqui;
- manter o nome original quando fizer sentido;
- se baixar uma nova versao, incluir data no nome do arquivo;
- registrar no script qual arquivo esta sendo lido.
